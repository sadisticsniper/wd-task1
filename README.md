MyCoffee – Coffee Shop Website

Overview -
MyCoffee is a responsive, modern website for showcasing coffee products, company history, and customer information. It’s designed to provide an engaging browsing experience with a clean layout, smooth navigation, and interactive elements. Perfect for small coffee shops or personal project portfolios.

Features -
1. Responsive Design: Works on desktop, tablet, and mobile screens.
2. Navigation Bar: Fixed vertical navbar with smooth scrolling to each section.
3. About Section: Highlights company history and story.
4. Product Section: Displays coffee products with images, pricing, and add-to-cart buttons.
5. Customer Section: Showcase satisfied customers or testimonials.
6. Footer: Social media links with interactive icons, support links, and company info.
7. Smooth Scrolling: Clicking navbar links scrolls to sections without awkward jumps.

Technologies Used -
1. HTML5 – Structure of the website.
2. CSS3 – Styling, layout, and responsive design.
3. Boxicons CDN – For social media and product icons.
4. CSS Variables – For color theming and easy customization.
5. Optional: Live Server (VS Code) for previewing changes locally.

How to Run Locally -

Clone the repository or download the project files.
Open index.html in your favorite browser.
(Optional) Use VS Code Live Server for live preview with auto-reload.
Click the navbar links to navigate between sections smoothly.
